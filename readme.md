## ğŸ“¦ **Installation**

### Prerequisites
- Node.js >= 18.0.0
- pnpm >= 8.0.0

### Setup
```bash
# Clone the repository
git clone https://github.com/eshanized/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
